# CSGO-Web-Server
This is a collection of console commands for CSGO legacy that turns the CSGO client into a webserver. The webserver hosts a controller allowing you to control CSGO from your browser

Check out this Youtube video for an absolutely horrendous explanation of what this is: https://youtu.be/g3HxUthCeQY

Just copy this into your autoexec.cfg file (you may have to remove the comments) and enable the netconport launch option for port 3001.

**DO NOT HOST THIS ONLINE** This is super insecure by design. This is for LAN play only.
